# Research

## CDSS-Research — ИИ-система для помощи медицинским специалистам. Система обрабатывает клинические рекомендации, создает возможности семантического поиска и предоставляет интеллектуальные ответы через чат-интерфейс.

Репозитории сервсисов:

   - Репозиторий серверной части (backend): https://github.com/citec-spbu/CDSS-Backend​
   - Репозиторий клиентской части (frontend): https://github.com/citec-spbu/CDSS-Frontend
   - Репозиторий для тестирования эмбеддингов и переранжирующих моделей: https://github.com/citec-spbu/CDSS-Embedding-Rerank-test​
   - Проектная доска с задачами и статусами их выполнения: https://github.com/orgs/CDSS-SPBU/projects/1
